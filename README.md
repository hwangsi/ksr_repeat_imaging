# 대한영상의학회 중복·재촬영 방지 활동 사이트

이 사이트는 별도의 빌드 과정 없이 그대로 배포되는 순수 HTML 사이트입니다.

## 파일 구성

| 파일/폴더 | 내용 |
|---|---|
| `index.html` | **화면에 보이는 모든 글이 이 파일에 있습니다.** 문구 수정은 여기서 하세요. |
| `styles.css` | 색상·글꼴·배치. 맨 위 `:root`의 색상 변수를 바꾸면 사이트 전체 색이 바뀝니다. |
| `script.js` | 메뉴·탭·팝업의 동작. 보통 수정할 일이 없습니다. |
| `assets/` | 로고 이미지 |
| `downloads/` | 자료실에서 내려받는 PDF·PPTX 원문 8건 |

## 글 수정하는 방법 (GitHub 웹에서)

1. GitHub에서 `index.html` 파일을 클릭
2. 오른쪽 위 연필(✏️) 아이콘 클릭
3. `Ctrl+F`로 바꾸려는 문장을 찾아 수정
4. 오른쪽 위 **Commit changes** 버튼 클릭

Vercel이 연결되어 있으면 저장 후 몇 초~1분 안에 사이트에 자동 반영됩니다.

주석(`<!-- ══════ … ══════ -->`)으로 구역이 표시되어 있어 원하는 부분을 쉽게 찾을 수 있습니다:
상단 메뉴 → 첫 화면 → 2025 통계 → 01 문제와 원칙 → 02 연구 연혁 → 03 사유코드 → 04 자료실 → 05 정책 제안 → 입장문 → 하단 정보 → 상세 모달(자료실 팝업 내용)

## 자료(PDF·PPTX) 추가·교체

1. 파일을 `downloads/` 폴더에 업로드
2. `index.html`에서 해당 자료의 카드와 모달 부분을 복사해 내용 수정
   (다운로드 링크 주소의 한글·특수문자는 URL 인코딩이 필요하니, 파일명은 가급적 영문·숫자로 하는 것이 편합니다)
